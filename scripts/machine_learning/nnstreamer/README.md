# i.MX NNStreamer Example Applications


These NNStreamer applications are examples on how to run NNStreamer
pipelines on i.MX devices. The application binaries are included in
build time with Yocto Project.

## Included examples

| Example | Description |
|----|----|
| [Image Classification](./classification/README.md) | Image classification is an ML task that attempts to comprehend an entire image as a whole. The goal is to classify the image by assigning it to a specific label. Typically, it refers to images in which only one object appears and is analyzed. |
| [Object Detection](./detection/README.md) | Object detection example using NNStreamer. Object detection is the ML task that detects instances of objects of a certain class within an image. A bounding box and a class label are found for each detected object. |
| [Pose Estimation](./pose/README.md) | Pose estimation example using NNStreamer. The goal of pose estimation is to detect the position and orientation of a person or object. In human pose estimation, this is usually done with specific keypoints such as hands, head, legs, etc. |
